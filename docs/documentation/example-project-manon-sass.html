Voorbeeldproject Manon en Sass

Stappenplan om een volledig nieuw project op te zetten met Manon en Sass.

Manon gebruiken binnen een project.

Benodigdheden om Manon succesvol te installeren binnen een nieuw project

  1. Project - Het project waarbinnen Manon gebruikt gaat worden
  2. NPM - Manon wordt aangeboden als een npm-package
  3. Sass - Manon maakt gebruik van Sass om de styling te builden.

Nieuw project opzetten

Projecten kunnen op veel manieren opgezet worden. Dit voorbeeld bevat een zo simpel mogelijke opzet om snel van start te kunnen. Dit kan altijd uitgebreid of omgebouwd worden.

  1. Maak een nieuwe map aan voor het nieuwe project.
  2. Voeg de benodigde basisbestanden toe aan het project. Bijvoorbeeld:
    • Een html-pagina. index.html
    • Een css-bestand voor het project. main.css
    • Een plek voor lettertypes, afbeeldingen en de css. Bijvoorbeeld een map static met daarbinnen de volgende mappen:
      • img
      • fonts
      • css
  3. Laad het css-bestand in via de html. Voeg de link naar het css bestand toe aan de head van alle html-bestanden. Voorbeeld: <link href="./static/css/main.css" rel="stylesheet" />

NPM installeren

  1. Controleer of npm aanwezig is op de computer. https://docs.npmjs.com/downloading-and-installing-node-js-and-npm#checking-your-version-of-npm-and-nodejs
  2. Installeer npm als deze nog niet aanwezig is op de computer. Volg de installatiestappen van npm https://docs.npmjs.com/downloading-and-installing-node-js-and-npm.
  3. Update npm indien nodig en gewenst. https://docs.npmjs.com/cli/v8/commands/npm-update.

Manon installeren

Voeg de npm-package van Manon toe aan het project:

  1. Locatie selecteren Navigeer in de terminal naar de map waar je Manon wilt toevoegen. Bijvoorbeeld de hoofdmap van het nieuwe project.
  2. Manon package installeren Installeer de package door het commando npm install @minvws/manon uit te voeren via de terminal in de geselecteerde map. De package op npm is te bekijken op: https://www.npmjs.com/package/@minvws/manon Deze stap is succesvol als de volgende bestanden automatisch zijn aangemaakt: package.json en de map node-modules aan binnen het project. Als de bestanden niet zijn aangemaakt voeg dan de NPM opzet handmatig toe.
  3. NPM opzet handmatig uitvoeren Deze stap overslaan als de bestanden package.json en node-modules als succesvol zijn aangemaakt. Voer het commando npm init uit in de map waar Manon geinstaleerd dient te worden. Er volgt een reeks vragen. Deze vragen kunnen ingevuld of overgeslagen worden. Als het bestand package.json nu aangemaakt is binnen het project kan de installatie van het pakket nogmaals uitgevoerd worden. Herhaal de stap Manon package installeren.
  4. Manon is geinstalleerd In de map node-modules/minvws/manon staan alle beschikbare componenten.

Manon gebruiken binnen het project

  1. Styling gebruiken Kies de gewenste componenten uit en importeer de css.
    1. Voeg een css bestand toe waarmee de benodigde css ingeladen gaat worden toe aan het project. Bijvoorbeeld binnen de css-map zoals deze aangemaakt is in Nieuw project opzetten. Noem deze bijvoorbeeld manon.css.
    2. Haal dit nieuwe css-bestand binnen in het project. Bijvoorbeeld door aan main.css een verwijzing toe te voegen. Bijvoorbeeld: @import "manon.css";
    3. Voeg de benodigde manon-componenten toe via het manon.css-bestand. Voorbeeld: @use "@minvws/manon/application-base";
    4. Voeg een css-bestand toe voor de variabelen. Bijvoorbeeld: variables.css
    5. Haal dit nieuwe css-bestand binnen in het project. Bijvoorbeeld door aan main.css een verwijzing toe te voegen. Bijvoorbeeld: @import "variables.css";
    6. Binnen het variabelenbestand kunnen de manon-component-variabelen toegevoegd en ingevuld worden.
    7. Let op: Manon maakt gebruik van Sass om de css te "builden". Voeg sass toe aan het project om de css-wijzigingen te zien.

Sass toevoegen

  1. Voeg Sass toe aan het project. Volg hiervoor bijvoorbeeld de installatiedocumentatie van Sass zelf. https://sass-lang.com/install
